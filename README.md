# minimal.nvim

## install

```sh
git clone git@github.com:fanlia/minimal.nvim.git ~/.config/nvim
```

## plugins
- nvim-telescope/telescope.nvim
- nvim-tree/nvim-tree.lua
- numToStr/Comment.nvim
